# Advent Of Code

## Utils

### New exercise

```
dotnet new console
```


### Run exercise

```
dotnet run
```

or 

To run without rebuilding (when testing performances):

```
dotnet run --no-build
```

### New exercise in F#

```
dotnet new console -lang "F#"
```

### Setup python env

```
pip3 install -r requirements.txt
```
